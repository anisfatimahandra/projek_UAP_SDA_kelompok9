## sistem perpustakaan


program ini adalah simulasi perpustakaan sederhana yang menyimpan daftar buku menggunakan Linked List.
Program dibuat dengan konsep OOP (Object-Oriented Programming) sehingga data dan
fungsi pengelolaan buku dibungkus di dalam sebuah class.

Konsep dalam project
- oop
- pointer
- linked list
- struct
- rekursif
